由Java撰寫的任何程式都需要運行在具有Java虛擬環境的電腦上，這也是Java的跨平台性質的來由，所以在運行任何Java程式之前都必須安裝Java Runtime Environment(JRE)，而Java Development Kit(JDK)則是Oracle公司提供給Java開發者配置Java程式開發環境的安裝套件，這個安裝套件本身就內建了JRE的部分，所以開發者只要安裝好JDK後，就可以開發或者行任何Java程式了。

本篇的重點將告訴讀者如何安裝JDK：
<blockquote class="bq_setting">
環境準備：<a style="color:black;" target="_blank" href="http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html">JDK 8</a>
</blockquote>

<blockquote class="bq_setting">步驟1：前往JDK 8官方網站，選擇下載與系統環境相符的安裝執行檔，小編的環境Windows x64，所以選擇jdk-8u144-windows-x64.exe。</blockquote>
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-JDK/1.JPG"></span>

步驟2：找到電腦中下載完成的安裝執行檔，雙點擊該安裝執行檔。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-JDK/2.JPG"></span>
 
步驟3：點擊Next開始設定安裝配置。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-JDK/3.JPG"></span>
 
步驟4：選擇並確認JDK的安裝路徑(預設路徑即可)後，點選Next開始執行安裝。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-JDK/4.JPG"></span>
 
步驟5：等待安裝程式安裝完成。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-JDK/5.JPG"></span>
 
步驟6：接著程式會進行JRE的安裝，選擇並確認JRE的安裝路徑(預設路徑即可)後，點選下一步開始執行安裝。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-JDK/6.JPG"></span>
 
步驟7：等待安裝程式安裝完成。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-JDK/7.JPG"></span>
 
步驟8：完成安裝後，按下Close完成安裝。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-JDK/8.JPG"></span>
