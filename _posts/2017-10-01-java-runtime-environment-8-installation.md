由Java撰寫的任何程式都需要運行在具有Java虛擬環境的電腦上，這也是Java的跨平台性質的來由，所以在運行任何Java程式之前都必須安裝Java Runtime Environment(JRE)，之後就可以開發或者行任何Java程式了。

本篇的重點將告訴讀者如何安裝JRE：

<blockquote class="bq_setting">
環境準備：<a style="color:black;" target="_blank" href="http://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html">Java Runtime Environment(JRE)</a>
</blockquote>

<div class="sp_setting">步驟1</div>
前往JRE官方網站，選擇下載與系統環境相符的安裝執行檔，小編的環境是Windows x64，所以選擇jre-8u144-windows-x64.exe。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-JRE/1.JPG"></span>

<div class="sp_setting">步驟2</div>
找到電腦中下載完成的安裝執行檔，雙點擊該安裝執行檔。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-JRE/2.JPG"></span>
 
<div class="sp_setting">步驟3</div>
點擊安裝(採用預設安裝路徑)。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-JRE/3.JPG"></span>

<div class="sp_setting">步驟3</div>
等待安裝程式完成。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-JRE/4.JPG"></span>

<div class="sp_setting">步驟3</div>
點擊關閉，接著就可以開啟任何Java程式了。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-JRE/5.JPG"></span>
