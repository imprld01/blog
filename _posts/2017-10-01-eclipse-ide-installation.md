Eclipse是一款為多數人使用的集成開發環境(Integrated Development Environment, IDE)。IDE能夠輔助開發者軟體的開發，IDE中內建相關語言的函式庫、除錯器、編譯器、直譯器與程式碼編輯器等子工具。使用IDE的好處在於開發者無須自行安裝多項輔助開發的工具，可以節省開發者配置開發環境的麻煩，IDE提供的圖形化介面也能提供開發者較直覺的功能使用。本篇將引導讀者安裝用來撰寫Java的IDE－Eclipse，當然Eclipse也支援多項語言的開發，包含C/C++、Python與PHP等程式語言。

本篇的重點將告訴讀者如何安裝開發Java的Eclipse IDE：

<blockquote class="bq_setting">
環境準備：
(1) <a style="color:black;" target="_blank" href="http://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html">Java Runtime Environment(JRE)</a>
(2) <a style="color:black;" target="_blank" href="http://www.eclipse.org/downloads/eclipse-packages/">Eclipse IDE</a>
</blockquote>

<div class="sp_setting">步驟1</div>
先行<a style="color:black;" target="_blank" href="http://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html">JRE</a>。

<div class="sp_setting">步驟2</div>
前往Eclipse官方網站，選擇下載與系統環境相符的安裝執行檔，小編的環境Windows x64，所以選擇64 bit。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-Eclipse/1.JPG"></span>

<div class="sp_setting">步驟3</div>
找到電腦中下載完成的安裝執行檔，雙點擊該安裝執行檔。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-Eclipse/2.JPG"></span>

<div class="sp_setting">步驟4</div> 
點擊欲安裝的套件，開始安裝，這裡選擇Eclipse IDE for Java Developers。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-Eclipse/3.JPG"></span>

<div class="sp_setting">步驟5</div>
選擇並確認JDK的安裝路徑(預設路徑即可)後，點選Install開始執行安裝。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-Eclipse/4.JPG"></span>

<div class="sp_setting">步驟6</div>
安裝程式完成後即可開始使用IDE。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-Eclipse/5.JPG"></span>

<div class="sp_setting">步驟7</div>
接著選擇並確認往後所有Java專案的存放路徑(預設路徑即可)後，點選Launch開始使用IDE。如果往後不需變更存放路徑，可以勾選Use this as the default and do not ask again，避免往後此視窗重複出現。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-Eclipse/6.JPG"></span>

<div class="sp_setting">步驟8</div>
等待載入完成後便可開始使用Eclipse開發自己的軟體了。
<span class="image featured"><img src="{{ site.baseurl }}/assets/img/20171001-Eclipse/7.JPG"></span>
